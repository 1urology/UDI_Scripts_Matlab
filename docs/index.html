Overview

Urodynamic investigations (UDI) combined with electromyography (EMG) allow simultaneous recording of bladder pressure, external urethral sphincter activity, and voided volumes in fully awake mice. This methodology enables detection of non-voiding contractions (NVCs) equivalent to detrusor overactivity in humans, and detrusor-sphincter dyssynergia (DSD) — key hallmarks of neurogenic lower urinary tract dysfunction. This Matlab pipeline performs automated analysis of UDI recordings from both TDMS (LabVIEW) and CSV data formats.

What the pipeline measures

ChannelSignalSampling
Intravesical pressureDetrusor muscle contractions500 Hz
EUS-EMGExternal urethral sphincter activity500 Hz

The pipeline supports research in spinal cord injury (SCI), partial bladder outlet obstruction (pBOO), and related models of lower urinary tract dysfunction (LUTD).

Data is saved as .tdms files (NI LabVIEW format). For experiments using the newer acquisition software, data is exported as paired pressure.csv and scale.csv files. Infusion rate is set to 20 µL/min for SCI experiments.

Analysis Pipeline

1

Load and preprocess

Read .tdms or .csv file. Extract pressure, scale, and EMG channels. Compute sampling rates. Normalize time vectors and align signals.

2

Visual inspection and cycle selection

Plot full normalized recording (pressure + scale). User clicks to select the start and end of a micturition cycle interactively (ginput).

3

Peak and threshold detection

Detect Pmax (peak pressure) and Pbase (minimum). Apply derivative-based threshold detection (65% of maximum dP/dt within the contraction window). Segment filling, contraction, and baseline phases.

4

NVC and contraction classification

Detect non-voiding contractions (NVCs) in the filling phase using a smoothed filtered signal. Classify peaks into five tiers: 15%, 5%, 2% above threshold, at threshold, sub-threshold, and microcontractions.

5

Compliance and voided volume

Calculate bladder compliance from filling duration and threshold pressure. Compute voided volume from positive scale changes over the whole cycle and the peak window.

6

Export figures and CSV results

Save publication-quality figures (overview, compliance window, NVC summary). Write all parameters to a results CSV file named by animal ID and recording timestamp.

NVC classification scheme

                      Pmax (voiding contraction peak)                        | -- 15% above threshold ----+---- NVC_15 strong non-voiding contractions -- 5% above threshold ----+---- NVC_5 -- 2% above threshold ----+---- NVC_2 -- Pthresh (voiding onset)-+---- NVC at-threshold contractions -- Sub-threshold ----------+---- sub-NVC -- Micro cutoff -----------+---- microcontractions (noise-adaptive)                        |                       Pbase (baseline pressure)
